# Data Directory

This directory contains processed data files for training and inference.

## Structure

```
data/
├── raw/           # Raw GDSC data files (not included, download separately)
└── processed/     # Processed PyTorch Geometric data (generated by preprocessing)
```

## Downloading Raw Data

Download the following files from the [GDSC database](https://www.cancerrxgene.org/downloads/bulk_download):

1. **Gene Expression Data**: `Cell_line_RMA_proc_basalExp.txt`
   - Source: Expression data section
   - Contains RMA-normalized basal gene expression for cancer cell lines

2. **Cell Line Details**: `Cell_Lines_Details.xlsx`
   - Source: Cell line annotations
   - Maps cell line names to COSMIC IDs

3. **Drug Annotations**: `screened_compounds_rel_8.5.csv`
   - Source: Compounds section
   - Contains drug pathway annotations

4. **IC50 Response Data**:
   - `GDSC1_fitted_dose_response_27Oct23.xlsx`
   - `GDSC2_fitted_dose_response_27Oct23.xlsx`
   - Source: Drug response section

Place these files in `data/raw/` directory.

## Generating Processed Data

After downloading raw data, run:

```bash
cd preprocessing
python preprocess_gdsc.py
```

This generates:
- `data/processed/gdsc_processed_train.pt`
- `data/processed/gdsc_processed_val.pt`
- `data/processed/gdsc_processed_test.pt`
- `data/processed/gdsc_processed_mappings.pt`
